软件模型架构与设计（校企合作课程）                                                                                                                                                                                                                                              
华为风扇控制系统项目案例-设计模式专题大作业                                                                                                                                                                                                                                      
项目内容：设计并实现了一个能根据实时状态自动调节风扇转速并上报告警的闭环控制系统，其核心功能涵盖风扇初始化、手动调速，并扩展至自动调速与业务温度告警。在核心架构设计中，运用单例模式对主控板（MasterControlBoard）进行建模，确保了控制核心的全局唯一性，从而统一管理和初始化下辖的多个风扇控制板，有效避免了配置冲突。为实现高级功能，引入观察者模式构建温度告警机制：将业务板（BusinessBoard）作为被观察者（Subject），当其温度状态变化时能主动通知所有告警观察者（AlarmObserver）。这种设计成功地将业务状态监控与告警处理逻辑解耦，显著提升了系统的可扩展性，整个过程严格遵循了面向对象的设计原则。
